# Anchor MPL Core

Metaplex Core functionality implementation including NFT collection management and creator whitelisting.

## Overview
This project implements Metaplex Core functionality on Solana, providing advanced NFT capabilities including collection management, creator verification, and digital asset standards.

## Features
- NFT collection management
- Creator whitelisting functionality
- Digital asset creation and management
- Collection authority operations

## Technologies
- Rust
- Anchor
- Solana CLI
- Metaplex
- SPL Tokens

## Getting Started
1. Install Solana CLI and Anchor
2. Run `anchor build` to build the program
3. Run `anchor test` to run integration tests
4. Deploy using `anchor deploy`
